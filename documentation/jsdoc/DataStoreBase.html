    abstract DataStoreBase(registry, name, cfgDir, saveCycle, defaults)

    Abstract class to be extended by the flat file DB classes. See Config and Database

    new abstract DataStoreBase(registry, name, cfgDir, saveCycle, defaults)
    Data/StoreBase.js, line 29

    This needs to be called in the extending class using super(registry, name, cfgDir, saveCycle, defaults).
    Parameters:
    Name Type Description
    registry Registry the core registry
    name string the name of the flat file
    cfgDir string the directory the flat file will be saved
    saveCycle number minimum interval in ms to save to disk
    defaults Object the default data to use when making a new file
