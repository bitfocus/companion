Supported operators include:

- Binary operators:
  - Addition: `a + b`
  - Subtraction: `a - b`
  - Multiplication: `a * b`
  - Division: `a / b`
  - Modulous: `a % b`
  - Exponentiation: `a ^ b`
- Unary operators:
  - Unary Negataion: `-a`
- Expression grouping:
  - Parenthesis: `(a + b) * c`

> **Note:** In the examples able `a` and `b` correspond to custom variables, dynamic variables or number literals. They are only used here for brevity.
