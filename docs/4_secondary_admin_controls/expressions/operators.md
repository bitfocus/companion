Supported operators include:

- Binary operators:
  - Addition: `a + b`
  - Subtraction: `a - b`
  - Multiplication: `a * b`
  - Division: `a / b`
  - Modulous: `a % b`
  - Equality (loose): `a == b`
  - Equality (strict): `a === b`
  - Inequality (loose): `a != b`
  - Inequality (stirct): `a !== b`
  - Greater than: `a > b`
  - Greater than or equal: `a >= b`
  - Less than: `a < b`
  - Less than or equal: `a <= b`
  - Logical OR: `a || b`
  - Logical AND: `a && b`
  - Right shift: `a >> b`
  - Left shift: `a << b`
  - Bitwise XOR: `a ^ b`
  - Bitwise AND: `a & b`
  - Bitwise OR: `a | b`
- Unary operators:
  - Unary Negataion: `-a`
  - Convert to number: `+a`
  - Logical NOT: `!a`
  - Bitwise NOT: `~a`
- Expression grouping:
  - Parenthesis: `(a + b) * c`

> **Note:** In the examples able `a` and `b` should be replaced with custom variables, module variables or number literals. They are only used here for brevity.
