Remote triggering can be done by sending RossTalk commands to port `7788`.

**Commands**

- `CC <page>/<row>/<column>`  
  _Press and release button_

**Examples**

Press and release row 0 column 5 on page 2  
`CC 2/0/5`

**Deprecated Commands**

- `CC <page>:<button>`  
  _Press and release button_
