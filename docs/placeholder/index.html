📚 Documentation

Docusaurus is not running in your environment

To work on or access the documentation in this mode, please run Docusaurus directly:

yarn dev:docs

The documentation will then be available at http://localhost:4000 as well as through the webui dev environment (yarn dev:webui).

Alternatively use one of:

yarn dev:webui:docs
yarn dev:webui:docs --base=/myurlbase

to start the Docusaurus server together with Vite (webui) and ensure they use the same BASE_URL.
Note: if starting from the webui folder, use yarn dev:docs instead.
